  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Images Still Not Loading?
  1. Kill all node: taskkill /F /IM node.exe
  2. Start backend first
  3. Wait 2 seconds
  4. Start frontend
  5. Hard refresh: Ctrl+Shift+R

ECONNREFUSED Errors?
  1. Backend not running on port 10000
  2. Check: netstat -ano | findstr 10000
  3. Restart backend

500 Errors?
  1. Check backend console for errors
  2. Verify MongoDB connection
  3. Restart backend
